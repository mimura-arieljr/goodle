# goodle

Web Application to serve as go-to reviewer for all sorts of subjects.

# Currently Supports

* AWS Cloud Certification Preparation Test 

## Pre-requisites

* Node Package Manager
* TailwindCSS
* Typescript

## Checkout

```bash
git clone 

```

## Running the program

Install dependencies:

```bash
npm install
```

Build dependencies:
```bash
npm run build
```

To update TailwindCSS dependencies:
```bash
npm run watch
```
