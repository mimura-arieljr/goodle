# goodle

Web Application to serve as go-to reviewer for all sorts of subjects.

# Currently Supports

* AWS Cloud Certification Preparation Test 

## Pre-requisites

* Node Package Manager
* TailwindCSS
* Typescript

## Checkout

```bash
git clone 

```

## Running the program

Install dependencies:
`npm install`

Build dependencies:
`npm run build`

To update TailwindCSS dependencies:
`npm run watch`
